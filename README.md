### Student Dashboard

- Heatmaps, line charts, pie charts  
- Instant scoring  
- Beautiful UI with transitions  

---

### Full-Screen Exam Protection
To prevent cheating:
- Quiz starts **only in full-screen mode**  
- If the student exits full-screen:  
  - Screen actions are blocked  
  - User must re-enter full-screen  
- If user exits **3 times â†’ Auto-submit exam**  
- Clean, strict, exam-mode environment
